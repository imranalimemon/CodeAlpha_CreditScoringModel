# Credit Scoring Model 💳

## CodeAlpha Machine Learning Internship — Task 1

### Overview
Predict an individual's **creditworthiness** (Good/Bad) using financial features like income, debt-to-income ratio, payment history, and credit history length.

### Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting (XGBoost-style)

### Evaluation Metrics
- Accuracy, Precision, Recall, F1-Score
- ROC-AUC Score
- 5-Fold Cross Validation

### Features
- Feature engineering from financial history
- ROC curve comparison across all models
- Confusion matrix analysis
- Feature importance visualization

### Tech Stack
- Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn

### How to Run
```bash
pip install -r requirements.txt
python credit_scoring.py
```
